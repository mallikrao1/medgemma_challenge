# Enterprise Roadmap

## Sprint 1 (Delivered)

- Multi-tenant auth + RBAC core
- Audit logging core
- API gateway middleware (request IDs + rate limits)
- Deployment starter manifests

## Sprint 2 (Delivered)

- Tenant-aware workflow APIs (templates/runs)
- Background job processing core (job queue + dispatcher + worker loop)
- Policy engine for safety/compliance checks
- Admin UI for policy/workflow/job operations

## Sprint 3

- SSO/OIDC + SCIM provisioning
- Fine-grained policy authorization (ABAC)
- Full observability stack (metrics/traces/log correlation)
- Data retention and archival workflows

## Sprint 4

- Multi-region DR strategy
- Advanced threat controls (WAF, anomaly detection)
- Formal compliance evidence packs (HIPAA/SOC2 readiness artifacts)
- Enterprise SLAs and incident runbooks
